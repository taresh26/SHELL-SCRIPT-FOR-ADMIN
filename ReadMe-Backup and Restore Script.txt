# Backup and Restore Script

## Overview
This script provides a generic solution for backing up and restoring directories on a Linux system. It creates a compressed archive of a specified directory and allows you to restore it from a backup file.

## Features
- **Backup:** Create a compressed backup of a directory.
- **Restore:** Restore a directory from a backup file.
