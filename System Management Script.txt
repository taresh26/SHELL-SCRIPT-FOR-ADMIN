# System Management Script

## Overview
This script provides a generic solution for managing system packages on Linux distributions. It supports Ubuntu (and other Debian-based distributions), Red Hat/CentOS, and SUSE. You can use it to:
- Identify installed packages
- Clean up unused packages
- Update package lists
- Upgrade installed packages
